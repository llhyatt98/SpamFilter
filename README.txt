Spam Detector

-----------------------------

Description: Our system is designed to allow users to determine whether a message is spam or not. This system uses the k-nearest neighbor algorithm and the perceptron algorithm to predict the result of input data.   

Authors: Lucas Hyatt and Olivia Pannell

How to run the code: 
1. Enter the following into the terminal "python3 ui.py"
2. The GUI must run with updated versions of Python (ideally Python 7 or newer)

How to use:
1. Check out User_Documentation.pdf

Software dependencies (Appendix):
	Python 3
	Numpy
	os
	collections
